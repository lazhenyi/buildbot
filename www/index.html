Welcome to Buildbot Dispatcher

GitHub Actions-style CI Dispatcher System

API Endpoints

Method Endpoint Description
GET /api/v1/info Dispatcher information
GET /api/v1/projects List all projects
GET /api/v1/builders List all builders
GET /api/v1/builds List builds
GET /api/v1/builds/{id} Build details
POST /api/v1/jobs/enqueue Enqueue a new job
GET /api/v1/jobs List dispatcher jobs
GET /api/v1/jobs/poll Runner polls for pending jobs
POST /api/v1/jobs/{id}/complete Mark job as complete
POST /api/v1/jobs/{id}/cancel Cancel a job
GET /api/v1/runners List registered runners
POST /api/v1/runners/register Register a new runner
POST /api/v1/runners/{id}/heartbeat Runner heartbeat
POST /api/v1/runners/{id}/unregister Unregister a runner
GET /api/v1/changes List source changes
GET /api/v1/steps/{build_id} Build steps
